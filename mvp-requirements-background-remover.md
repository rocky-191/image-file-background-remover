## 1. 产品概述

### 1.1 产品定位
一款在线图片背景去除工具，帮助用户快速移除图片背景，获得透明背景或纯色背景的图片。

### 1.2 目标用户
| 用户类型 | 使用场景 |
|---------|---------|
| 电商卖家 | 商品图抠图、制作主图 |
| 设计师 | 素材处理、创意设计 |
| 社交媒体用户 | 头像、封面图处理 |
| 普通用户 | 日常图片处理 |

### 1.3 核心价值
- **快速**：上传即处理，几秒完成
- **简单**：拖拽上传，无需学习成本
- **免费**：每天免费3次，无门槛使用

## 4. 功能详细说明

### 4.1 F01 - 图片上传

**描述**：用户上传需要处理的图片

**交互方式**：
- 拖拽上传（拖拽图片到上传区域）
- 点击上传（点击按钮选择文件）

**支持格式**：JPG、PNG、WebP

**限制**：
- 最大文件大小：10MB
- 最小尺寸：50x50 像素
- 最大尺寸：8000x8000 像素

**前端校验**：
```
✓ 文件类型是否支持
✓ 文件大小是否超限
✓ 图片尺寸是否符合
```

**异常处理**：
- 格式不支持 → 提示"仅支持 JPG、PNG、WebP 格式"
- 文件过大 → 提示"图片大小不能超过 10MB"
- 读取失败 → 提示"图片读取失败，请重试"

---

### 4.2 F02 - 背景去除

**描述**：调用 Remove.bg API 去除图片背景

**处理流程**：
```
1. 前端将图片转为 Base64
2. 发送给 Cloudflare Workers
3. Workers 调用 Remove.bg API
4. 返回结果（PNG 透明背景）
5. 前端展示
```

**API 调用**：
```javascript
// 请求
POST https://api.remove.bg/v1.0/removebg
X-Api-Key: [API_KEY]
Image: [Base64/File]

// 响应
Content-Type: image/png
[PNG 二进制数据]
```

**超时处理**：
- 处理时限：25秒
- 超时提示："处理超时，请尝试更小的图片"

---

### 4.3 F03 - 结果预览

**描述**：展示原图与处理结果对比

**布局**：
```
┌─────────────┬─────────────┐
│             │             │
│    原图     │    结果     │
│             │             │
└─────────────┴─────────────┘
       ◀              ▶
    滑动对比 / 切换查看
```

**功能**：
- 左右滑动对比
- 鼠标悬停查看结果
- 缩放查看细节

---

### 4.4 F04 - 图片下载

**描述**：下载处理后的图片

**下载格式**：PNG（透明背景）

**文件名规则**：
```
remove-bg-{timestamp}.png
例：remove-bg-20260315122000.png
```

**下载方式**：
- 点击按钮直接下载
- 右键图片另存为

---

### 4.5 F05 - 免费次数限制

**描述**：控制免费用户的使用次数

**规则**：
| 角色 | 每日免费次数 | 周期 |
|------|-------------|------|
| 访客 | 3次 | 每天 00:00 重置 |

**计数逻辑**：
```
1. 用户首次访问，生成临时 ID
2. 每次处理扣除 1 次
3. 当天达到上限，显示充值提示
4. 次日 00:00 自动重置
```

**提示文案**：
- 剩余 1 次："今日还可免费处理 1 次"
- 剩余 0 次："今日免费次数已用完，付费继续使用"

---

### 4.6 F06 - 处理进度提示

**描述**：让用户了解处理状态

**状态流转**：
```
[空闲] → [上传中] → [处理中] → [完成/失败]
```

**提示文案**：
| 状态 | 文案 |
|------|------|
| 上传中 | 正在上传图片... |
| 处理中 | 正在去除背景...（约3-5秒）|
| 完成 | 处理完成！ |
| 失败 | 处理失败，请重试 |
